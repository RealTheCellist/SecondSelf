# Second Self 화면 카피 초안

## 문서 목적

이 문서는 `Second Self` 7일 체크인 실험용 MVP의 핵심 화면 카피 초안이다.

목표는 다음 두 가지다.

- 사용자가 첫 10초 안에 무엇을 해야 하는지 이해하게 만든다
- 결과 화면에서 `기록 앱`이 아니라 `나를 비춰주는 도구`처럼 느끼게 만든다

## 카피 원칙

- 짧고 선명하게 쓴다
- 설명보다 참여를 유도한다
- 과장된 치유/진단 어조를 피한다
- 자기계발 문구보다 자기 해석 문구에 가깝게 쓴다

## 톤 가이드

- 차분하다
- 해석적이다
- 단정적이지 않다
- 약간의 여백이 있다

좋은 예:

- 요즘의 나는 무엇에 자주 흔들리고 있을까?
- 최근의 나를 짧게 비춰보세요.
- 당신의 입력을 한 줄의 해석으로 돌려드립니다.

피해야 할 예:

- 당신의 진짜 문제를 분석해드립니다
- 지금 당장 삶을 바꿔보세요
- 당신의 감정을 정확히 진단합니다

## 화면 1. 첫 질문 선택 화면

### 안 A

- 헤드라인: 오늘의 나는 어디에 가장 오래 머물렀나요?
- 서브카피: 한 가지 질문을 고르고, 짧게 적어보세요.

질문 카드:

- 요즘 자주 붙잡히는 생각은 무엇인가요?
- 최근 반복해서 느끼는 감정은 무엇인가요?
- 지금의 나를 한 문장으로 표현하면 어떤가요?

버튼:

- 시작하기

### 안 B

- 헤드라인: 최근의 나를 가장 잘 건드리는 질문 하나
- 서브카피: 설명보다 먼저, 지금의 마음 가까이에서 시작합니다.

질문 카드:

- 머릿속에 자주 맴도는 생각이 있나요?
- 요즘 자주 돌아오는 감정이 있나요?
- 지금의 나를 짧게 말하면 어떤가요?

버튼:

- 이 질문으로 시작

### 안 C

- 헤드라인: 기록보다 먼저, 지금의 나를 골라보세요
- 서브카피: 오래 쓰지 않아도 됩니다. 한 문장으로 충분합니다.

질문 카드:

- 내가 요즘 자주 붙잡는 생각
- 내가 최근 반복해서 느끼는 감정
- 지금의 나를 표현하는 한 문장

버튼:

- 선택 완료

## 화면 2. 체크인 작성 화면

### 공통 카피

- 상단 라벨: 오늘의 체크인
- 보조 문구: 30초면 충분합니다.

입력 필드 플레이스홀더:

- 떠오르는 그대로 적어보세요
- 정리되지 않아도 괜찮아요
- 오늘 자주 붙잡혔던 생각부터 적어보세요

감정 태그 섹션:

- 라벨: 지금 가까운 감정을 골라보세요
- 보조 문구: 1개 또는 2개만 선택해도 충분합니다

완료 버튼:

- 이대로 비춰보기
- 해석 받기
- 오늘 체크인 완료

## 화면 3. 첫 결과 화면

### 해석 1줄 영역

헤드라인 후보:

- 오늘의 해석
- 지금의 나를 비추면
- 방금 남긴 생각을 정리하면

해석 문장 예시:

- 요즘 당신은 결과보다 방향을 더 오래 붙잡고 있습니다.
- 최근의 당신은 감정보다 정리되지 않은 생각에 더 머무르고 있습니다.
- 지금의 당신은 확신보다 정돈에 더 에너지를 쓰고 있습니다.

보조 카피 후보:

- 완전히 맞지 않아도 괜찮습니다. 지금의 나를 가볍게 비춰보는 중이니까요.
- 이 문장은 진단이 아니라, 오늘의 입력을 정리한 한 줄입니다.
- 지금의 흐름을 짧게 되돌려드렸어요.

CTA 후보:

- 내 패턴 보기
- 내일 다시 체크인하기
- 오늘의 흐름 저장

## 화면 4. 패턴 피드 화면

### 섹션 카피

- 메인 헤드라인: 최근의 나는 이렇게 반복되고 있어요
- 서브카피: 최근 5번의 체크인에서 자주 나온 흐름입니다

카드 1:

- 제목: 자주 나온 주제
- 예시 값: 일, 방향성, 관계

카드 2:

- 제목: 반복 감정
- 예시 값: 불안, 압박감

카드 3:

- 제목: 최근의 나
- 예시 값: 요즘 당신은 성과보다 앞으로의 방향을 더 자주 고민하고 있습니다

하단 CTA:

- 오늘 다시 돌아보기
- 다음 체크인 남기기

## 빈 상태 카피

### 첫날 또는 데이터 부족 상태

- 헤드라인: 아직은 오늘의 한 줄이 더 정확합니다
- 본문: 체크인이 몇 번 더 쌓이면, 반복되는 감정과 주제를 함께 보여드릴게요.

### 체크인 없음

- 헤드라인: 오늘의 한 줄부터 시작해볼까요?
- 본문: 길게 쓰지 않아도 됩니다. 짧은 입력 하나면 충분합니다.

## 마이크로카피

- 저장 완료: 오늘의 체크인이 반영되었어요
- 태그 미선택 경고: 감정 태그를 1개 이상 골라주세요
- 텍스트 미입력 경고: 한두 문장만 적어도 충분합니다
- 네트워크 오류: 잠시 연결이 불안정해요. 다시 한번 시도해주세요

## 추천안

첫 실험 버전에서는 아래 조합을 추천한다.

- 첫 질문 선택 화면: 안 A
- 체크인 완료 버튼: `이대로 비춰보기`
- 첫 결과 헤드라인: `지금의 나를 비추면`
- 패턴 피드 헤드라인: `최근의 나는 이렇게 반복되고 있어요`
