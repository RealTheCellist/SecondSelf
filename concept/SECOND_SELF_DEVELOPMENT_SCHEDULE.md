## 주간 우선순위

### 반드시 끝낼 것

- 첫 질문 선택 -> 체크인 -> 결과 화면 흐름 안정화
- 해석 1줄 품질 보강
- 패턴 카드 3장 정상 노출
- 기본 QA 완료

### 되면 좋은 것

- 이벤트 로깅 정리
- 프리뷰 상태 추가
- 히스토리 표현 개선

### 다음 주로 넘겨도 되는 것

- 프레임워크 전환
- 서버 저장소 연결
- 고급 AI 해석
- 음성 입력

## 리스크

- 해석 문장이 반복되면 금방 기계적으로 보일 수 있다
- 규칙형 로직만으로는 입력 품질 편차를 다 흡수하기 어렵다
- 사용자 테스트 전에 문구를 너무 많이 다듬으면 오히려 실제 반응을 놓칠 수 있다
