# Second Self 7일 체크인 실험용 PRD

## 문서 목적

이 문서는 `Second Self`의 첫 MVP를 7일 실험 단위로 검증하기 위한 제품 요구사항 정의서다.

목표는 완성형 자기이해 서비스 구축이 아니라, 아래 핵심 가설을 짧은 사이클 안에서 검증하는 것이다.

- 사용자는 짧은 체크인 후 돌아오는 `해석 1줄`에서 가치를 느끼는가
- 사용자는 기록 자체보다 `최근의 나를 다시 확인하기 위해` 재방문하는가
- 규칙형 패턴 요약만으로도 저널링 앱과 다른 감각을 줄 수 있는가

## 실험 한 줄 정의

사용자가 7일 동안 하루 1회 짧게 체크인하고, 최근 5개 입력을 바탕으로 자신의 반복 감정과 주제를 확인하는 텍스트 기반 프로토타입.

## 배경

`Second Self`의 핵심은 기록이 아니라 반사다.  
따라서 이번 실험은 입력 수단을 넓히는 것이 아니라, 가장 작은 입력으로도 사용자가 `맞다 싶은 한 줄`을 받는 경험을 만들 수 있는지 검증하는 데 집중한다.

## 목표

### 비즈니스 목표

- `Second Self`가 단순 저널링 앱이 아니라는 초기 신호를 확보한다
- 작은 MVP 범위로도 재방문 이유를 만들 수 있는지 확인한다

### 사용자 목표

- 짧은 시간 안에 현재의 나를 돌아볼 수 있다
- 최근 반복되는 감정과 주제를 확인할 수 있다
- 입력한 내용이 정리되어 돌아오는 느낌을 받는다

### 실험 목표

- 첫 체크인 완료율 검증
- 첫 결과 화면 가치 이해도 검증
- 7일 내 재방문 및 반복 체크인 가능성 검증

## 제외 목표

- 정교한 감정 분석 정확도 검증
- 장기 아카이브 사용성 검증
- 음성 입력 수요 검증
- 고급 AI 코칭 효과 검증
- 복잡한 개인화 추천 검증

## 타깃 사용자

- 커리어 전환기 사용자
- 삶의 방향을 재정리하는 사람
- 자기 성찰 욕구가 높은 직장인
- 감정과 동기를 짧게 정리하고 싶은 사용자

공통 특성:

- 긴 저널링 습관은 없어도 된다
- 짧고 가벼운 자기 점검에는 열려 있다
- 생산성 툴보다 자기 이해 도구에 더 반응한다

## 핵심 가설

### 가설 1

첫 진입에서 설명보다 `질문 선택 -> 짧은 입력 -> 해석 1줄` 흐름을 주면 사용자는 제품 가치를 더 빠르게 이해한다.

### 가설 2

최근 5개 입력 기준의 규칙형 패턴 요약만으로도 사용자는 `나를 비춰주는 느낌`을 받을 수 있다.

### 가설 3

사용자는 7일 동안 하루 1회 수준의 가벼운 체크인에는 부담 없이 참여할 수 있다.

### 가설 4

사용자의 재방문 이유는 기록 누적보다 `내 상태를 다시 확인하고 싶어서`에 더 가깝게 나타난다.

## 제품 원칙

- 텍스트 전용으로 시작한다
- 첫 세션은 1분 이내로 설계한다
- 첫 화면은 소개보다 참여가 먼저다
- 결과는 길게 설명하지 않고 짧고 선명하게 제공한다
- 규칙형 요약으로 먼저 검증하고, 고급 AI 해석은 후순위로 둔다

## 사용자 시나리오

### 시나리오 A

커리어 전환을 고민하는 사용자가 앱에 들어와 `요즘 자주 붙잡히는 생각은 무엇인가요?`를 선택하고 짧게 답한다.  
입력 직후 `최근의 나는 방향성에 더 오래 머무는 편`이라는 해석 1줄을 보고, 다음 며칠 동안 같은 질문 또는 다른 질문으로 다시 들어온다.

### 시나리오 B

감정 기복을 정리하고 싶은 사용자가 하루 1회 체크인을 하며 감정 태그를 선택한다.  
5일차 이후 `반복 감정: 불안, 압박감` 카드와 함께 `성과보다 방향을 더 고민하고 있음` 같은 요약을 확인하고, 스스로 패턴을 인식한다.

## 핵심 사용자 흐름

1. 진입
2. 질문 1개 선택
3. 짧은 텍스트 입력
4. 감정 태그 1~2개 선택
5. 즉시 해석 1줄 + 키워드 2개 확인
6. 체크인이 누적되면 패턴 카드 3장 확인
7. 다음 날 다시 체크인

## 기능 범위

### 포함 기능

- 하루 1회 체크인 생성
- 고정 질문 3개 중 1개 선택
- 자유 텍스트 입력 1개
- 감정 태그 1~2개 선택
- 체크인 직후 해석 1줄 노출
- 최근 5개 입력 기준 패턴 카드 3장 노출
- 7일 실험 기준 최소 히스토리 조회

### 제외 기능

- 음성 입력
- 무제한 과거 기록 탐색
- 문장 단위 편집 기능
- 소셜 공유
- AI 대화형 코칭
- 푸시 알림 최적화
- 복잡한 계정/커뮤니티 기능

## 정보 구조

목적:

- 사용자가 고민 없이 바로 입력을 시작하게 만든다

구성 요소:

- 상단 질문 선택 영역
- 3개 질문 카드
- 자유 텍스트 입력 필드
- 감정 태그 선택 영역
- 완료 버튼

상태:

- 입력 전
- 입력 중
- 제출 가능
- 제출 완료

### 화면 2. 결과 및 패턴 피드 화면

목적:

- 사용자가 입력한 내용을 짧고 의미 있게 되돌려준다

구성 요소:

- 오늘의 해석 1줄
- 오늘의 키워드 2개
- 최근 자주 나온 주제 카드
- 반복 감정 카드
- 최근의 나 요약 카드

상태:

- 첫 체크인 직후: 해석 1줄 중심
- 체크인 누적 후: 패턴 카드 3장 함께 노출

## 질문 세트

초기 질문은 고정 3개로 시작한다.

1. 요즘 자주 붙잡히는 생각은 무엇인가요?
2. 최근 반복해서 느끼는 감정은 무엇인가요?
3. 지금의 나를 한 문장으로 표현하면 어떤가요?

선정 원칙:

- 추상적이되 부담스럽지 않아야 한다
- 30초 안에 답할 수 있어야 한다
- 감정과 주제를 자연스럽게 끌어내야 한다

## 감정 태그

초기 태그는 8~12개 수준으로 제한한다.

예시:

- 불안
- 압박감
- 기대
- 자신감
- 혼란
- 안정감
- 의욕
- 무기력

원칙:

- 너무 세밀하게 나누지 않는다
- 사용자가 빠르게 고를 수 있어야 한다
- 패턴 집계에 바로 활용 가능해야 한다

## 패턴 요약 로직

이번 실험에서는 생성형 AI 중심 해석 대신 규칙형 템플릿을 사용한다.

### 입력 데이터

- 체크인 날짜
- 선택 질문 ID
- 자유 텍스트
- 감정 태그 1~2개

### 집계 기준

- 최근 5개 체크인 기준
- 반복 등장 단어 추출
- 감정 태그 빈도 집계
- 문장 내 반복 표현 또는 주제성 단어 확인

### 출력 규칙

- 카드 1: 최근 자주 나온 주제
- 카드 2: 반복 감정
- 카드 3: 최근의 나 요약

### 출력 예시

- 최근 자주 나온 주제: 일, 방향성, 관계
- 반복 감정: 불안, 압박감
- 최근의 나: 요즘 당신은 성과보다 앞으로의 방향을 더 자주 고민하고 있습니다

## 해석 1줄 가이드

원칙:

- 단정적이지 않게 쓴다
- 과장하지 않는다
- 사용자의 언어를 정리해 되돌려주는 느낌을 준다
- 치료나 진단처럼 읽히지 않게 한다

좋은 예:

- 요즘 당신은 결과보다 방향을 더 자주 점검하고 있습니다
- 최근의 당신은 관계보다 일의 흐름에 더 민감하게 반응하고 있습니다
- 지금의 당신은 확신보다 정리에 더 에너지를 쓰고 있습니다

피해야 할 예:

- 당신은 불안 장애 경향이 있습니다
- 당신의 진짜 문제는 관계 회피입니다
- 당신은 반드시 커리어 전환을 해야 합니다

## 성공 지표

### 정량 지표

- 첫 체크인 완료율
- 첫 결과 화면 도달율
- 7일 내 3회 이상 체크인 비율
- 7일 내 5회 이상 체크인 비율
- 결과 화면 재방문 비율

### 정성 지표

- "맞다" 또는 "생각해볼 만하다" 반응 비율
- "기록 앱 같다"보다 "나를 비춰준다"는 반응 비율
- 재방문 이유가 `기록`보다 `상태 확인`에 가까운지 여부

## 실험 성공 기준

아래 기준을 모두 만족하면 다음 단계로 진행한다.

- 첫 체크인 완료율이 충분히 높다
- 첫 결과 화면에서 가치 이해가 분명히 나타난다
- 실험 참여자 중 의미 있는 비율이 7일 내 반복 체크인을 한다
- 정성 인터뷰에서 `반사`, `비춰준다`, `정리된다` 계열 반응이 확인된다

수치 기준은 테스트 모집 방식에 따라 달라질 수 있으므로, 초기에는 절대값보다 상대 비교와 정성 해석을 함께 본다.

## 리스크

- 첫 결과가 빈약하면 저널링 앱처럼 느껴질 수 있다
- 5개 미만 데이터 구간에서 패턴 카드가 약하게 보일 수 있다
- 규칙형 해석 문장이 반복되면 기계적으로 느껴질 수 있다
- 질문이 너무 추상적이면 입력 시작 자체가 어려울 수 있다

## 오픈 이슈

1. 첫날에도 패턴 카드를 일부 보여줄지, 해석 1줄만 보여줄지
2. 감정 태그를 필수로 할지 선택으로 둘지
3. 최근 5개 기준이 적절한지, 최근 3개 기준이 더 나은지
4. 7일 실험에서 리마인드 수단을 둘지 말지
