# Second Self

## 한 줄 정의

짧은 체크인을 통해, 요즘의 내가 반복해서 붙잡는 감정과 주제를 비추어주는 정체성 탐색형 개인 서비스.

## 제품 한 줄 약속

기록을 쌓게 하는 앱이 아니라, 최근의 나를 한 줄로 되비춰주는 앱.

## 문제 정의

많은 사람들은 자기 자신을 더 이해하고 싶어 하지만, 기존 저널링 앱과 습관 앱은 기록 자체에 머무는 경우가 많다. 사용자는 글을 남겨도 결국 `나는 요즘 무엇에 반복해서 흔들리고 있는가`, `어떤 방향으로 생각이 쏠리고 있는가`를 바로 확인하기 어렵다.

## 핵심 사용자

- 커리어 전환기 사용자
- 삶의 방향을 재정리하는 사람
- 자기 성찰 욕구가 높은 직장인
- 감정과 동기를 정리하고 싶은 사용자

## 핵심 가치

- 최근의 나를 더 선명하게 이해할 수 있다
- 반복되는 감정과 주제를 확인할 수 있다
- 기록이 아니라 자기 해석의 단서를 얻을 수 있다

## 차별점

`Second Self`는 기록 도구보다 `정체성 소프트웨어`에 가깝다.

차별점의 핵심은 입력이 아니라 반사다.

- 사용자가 무엇을 남겼는지보다
- 서비스가 그 안에서 무엇을 되비춰주는지가 더 중요하다

즉, 저널링 앱처럼 쌓는 경험이 아니라 `나의 최근 패턴을 비추는 거울`처럼 느껴져야 한다.

## 회의 반영 결론

에이전트 회의 기준으로, 이 제품의 MVP는 `체크인 기능` 자체가 아니라 `첫 요약에서 맞다 싶은 아하 경험`을 만드는 것이다.

따라서 초기 버전은 기능을 넓히기보다 다음 질문에 답해야 한다.

- 사용자가 첫 1분 안에 무엇을 하게 할 것인가
- 첫 세션 끝에 어떤 결과를 보여줄 것인가
- 그 결과가 왜 저널링 앱과 다르게 느껴지는가

상세 논의는 `SECOND_SELF_AGENT_MEETING.md`를 따른다.

## 초기 MVP 원칙

- 텍스트 전용으로 시작한다
- 첫 세션은 1분 이내로 끝나야 한다
- 첫 화면은 소개보다 참여가 먼저여야 한다
- 결과는 길게 설명하지 않고 짧고 선명해야 한다
- 요약은 과도한 AI 해석보다 규칙형 템플릿으로 시작한다

## 초기 MVP 범위

- 하루 1회 텍스트 체크인
- 고정 질문 3개 중 1개 선택
- 자유 텍스트 입력 1개
- 감정 태그 1~2개 선택
- 최근 5개 입력 기준 패턴 카드 3장 제공
- 해석 1줄 제공

## 제외 범위

- 음성 입력
- 복잡한 감정 분석
- 긴 아카이브 편집 기능
- 소셜 공유
- 고급 AI 코칭
- 초기부터 큰 서사형 정체성 리포트 제공

## 첫 세션 목표

첫 세션의 목표는 사용자가 깊이 분석받았다고 느끼게 하는 것이 아니다.  
`이 서비스가 나를 꽤 정확하게 비춰줄 수 있겠다`는 감각을 만드는 것이 목표다.

그래서 첫 세션은 다음 기준을 만족해야 한다.

- 무엇을 해야 하는지 즉시 이해된다
- 입력 부담이 작다
- 끝났을 때 한 줄 해석이 돌아온다
- 사용자가 "맞다" 또는 "생각해볼 만하다"라고 반응할 수 있다

## 추천 온보딩 구조

첫 화면은 제품 소개 화면보다 `첫 질문 선택 화면`이 적합하다.

예시 질문:

1. 요즘 자주 붙잡히는 생각은 무엇인가요?
2. 최근 반복해서 느끼는 감정은 무엇인가요?
3. 지금의 나를 한 문장으로 표현하면 어떤가요?

추천 흐름:

1. 질문 1개 선택
2. 30초 내 짧은 텍스트 입력
3. 감정 태그 1~2개 선택
4. 즉시 해석 1줄 + 키워드 2개 확인

## 핵심 사용자 흐름

1. 첫 질문 선택
2. 짧은 텍스트 체크인 작성
3. 감정 태그 선택
4. 현재 입력에 대한 해석 1줄 확인
5. 체크인이 누적되면 최근 5개 기준 패턴 카드 확인

## MVP 화면 구조

### 1. 체크인 화면

- 질문 선택
- 짧은 텍스트 입력
- 감정 태그 선택

### 2. 패턴 피드 화면

- 최근 자주 나온 주제
- 반복 감정
- 해석 1줄

초기 MVP에서는 이 2개 화면만으로도 충분하다.

## 패턴 요약 로직 최소안

초기에는 생성형 AI 중심 해석보다 규칙형 요약으로 시작한다.

입력 데이터:

- 선택한 질문
- 자유 텍스트 1개
- 감정 태그 1~2개

집계 방식:

- 최근 5개 입력에서 반복 등장 단어 추출
- 반복 감정 태그 집계
- 자주 나오는 문장 패턴 확인

출력 방식 예시:

- 최근 자주 나온 주제: 일, 방향성, 관계
- 반복 감정: 불안, 압박감
- 한 줄 해석: 요즘 당신은 성과보다 앞으로의 방향을 더 자주 고민하고 있습니다

## 메시지 가이드

- 첫 인상에서 기록 앱처럼 보이면 안 된다
- `감정 분석`보다 `최근의 나를 비춰준다`는 표현이 적합하다
- `자기계발`보다 `자기 해석`에 가까운 톤이 맞다

권장 메시지 예시:

- 요즘의 나는 무엇에 자주 흔들리고 있을까?
- 기록을 쌓기보다, 최근의 나를 비춰보세요.
- 당신의 생각과 감정 속 반복 패턴을 짧고 선명하게 되돌려드립니다.

## 주요 리스크

- 가치가 추상적으로 들리면 저널링 앱의 또 다른 변주로 보일 수 있다
- 첫 세션에서 반복 패턴을 과하게 약속하면 신뢰를 잃을 수 있다
- 결과 문장이 과도하게 거창하면 실제 데이터 밀도보다 과장돼 보일 수 있다

## 성공 기준

- 첫 체크인 완료율이 높다
- 첫 결과 화면에서 서비스 가치를 바로 이해한다
- 7일 이내 재방문 이유가 기록 자체보다 `내 상태를 다시 확인하고 싶어서`로 나타난다
